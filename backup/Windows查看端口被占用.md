## Windows下查看哪个端口被哪些进程占用，有时候开启软件服务的时候提示端口占用。

在Windows命令提示符下输入以下命令：
```
 netstat -ano        //列出所有端口
```
![1](https://img2020.cnblogs.com/blog/2034475/202012/2034475-20201209112945987-528824134.png)

查找具体的某个端口命令：
```
netstat -ano | findstr 3389        //查找3389端口
```
