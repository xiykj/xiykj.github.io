## 最近想把放在抽屉的树梅派拿来使用练手，在不用的话板子都要坏了，想在树梅派上听歌，试用了网易云音乐/mplayer/qmmp播放器

### 还是觉得qmmp比较好用，网易云音乐因播放不了本地音乐直接pass

　　安装qmmp播放器：
```
　　sudo apt-get update　　　　　　更新源列表
　　sudo apt-get qmmp　　　　　　安装qmmp播放器
　　sudo apt-get mpg123　　　　　　　安装音乐解码器
```
